# Домашние задания по C#

> [!IMPORTANT]
> Каждая программа — самодостаточная консольная утилита: можно запускать независимо и проверять вводом с клавиатуры.  
> Для простоты все решения лежат отдельными .cs-файлами в корне.

- [Факториал с проверкой переполнения](./FactorialWithCheck.cs) — вычисление в checked+catch OverflowException.
- [Рекурсивный факториал](./RecursiveFactorial.cs) — корректно для диапазона long.
- [НОД двух и трёх чисел](./GCDCalculation.cs) — алгоритм Евклида; перегрузка для трёх аргументов.
- [Рекурсивный Фибоначчи](./FibonacciRecursive.cs) — F(n-1)+F(n-2) с базами 1 и 2.
- [Обмен элементов массива](./ArrayElementSwap.cs) — обмен по значениям, поиск индексов через Array.IndexOf.
- [Работа с параметрами](./ParamsExample.cs) — params + ref/out: сумма, произведение, среднее.
- [Цифры как ASCII‑рисунки](./DigitalArt/) — вывод 0–9 псевдографикой; обработка exit/закрыть и ошибок ввода.
- [Структуры: проверка «нецензурной лексики»](./GrandpaSwearCheck.cs) — enum уровней, struct Grandpa, метод CheckSwearing(ref,…).

## Как запускать

> [!NOTE]
> Ниже показано два удобных способа: через проект или, при новой версии SDK, напрямую по файлу.

- Вариант A (универсальный): создать консольный проект, заменить Program.cs и запустить:
  1) `dotnet new console -n Sandbox`  
  2) заменить `Sandbox/Program.cs` содержимым нужного файла  
  3) `cd Sandbox && dotnet run`
- Вариант B (если поддерживается в вашей версии SDK): выполнить файл напрямую:  
  `dotnet run --file MaxNumber.cs`
